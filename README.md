# Quiet Feed

Shows Facebook and Vk feeds as empty space without any information on it.

# Installation

Set `User Style Sheet` or `User CSS` in your browser to `quiet-feed.css`.

* **Safari**: `Preferences...` → `Advanced` tab → `Style sheet` dropdown menu.
